# Tinaplayer | a Lightweight PWA Audio Player

Live: https://player-d70.pages.dev/

![screenshot1.png](public%2Fscreenshot1.png)

## Libraries
- React
- Zustand
- Tailwind Css
- Material tailwind
- vite
- buffer
- idb
- music-metadata
- music-metadata-browser
- Eslint
- Prettier
- Husky

## How To Run
`
yarn add
`
`
yarn run dev
`
`
yarn run build
`
